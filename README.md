# 🥋 Arena Jiu Jitsu - Sistema de Controle de Vendas

Sistema desenvolvido para gerenciamento de vendas, alunos, mensalidades, estoque e relatórios da **Arena Jiu Jitsu**, oferecendo uma interface moderna, rápida e intuitiva para o controle financeiro e administrativo da academia.

---

## 📋 Funcionalidades

### 👥 Gestão de Clientes
- Cadastro de alunos e clientes
- Registro de telefone
- Cadastro de CPF
- Controle de matrículas
- Consulta rápida de clientes

### 💰 Controle de Vendas
- Registro de vendas
- Associação de vendas a clientes
- Histórico completo
- Controle de formas de pagamento:
  - PIX
  - Dinheiro
  - Cartão

### 📦 Gestão de Produtos
- Cadastro de produtos
- Controle de estoque
- Código de barras
- Preço unitário
- Atualização automática do estoque

### 🏷️ Categorias Personalizadas
- Cadastro de novas categorias
- Edição de categorias existentes
- Exclusão de categorias
- Organização automática dos produtos

### 🥋 Planos da Academia
- Plano Mensal
- Plano Trimestral
- Plano Anual
- Controle de vencimentos

### 📊 Dashboard Administrativo
- Total de vendas
- Total faturado
- Produtos cadastrados
- Clientes cadastrados
- Visão geral do sistema

### 📈 Relatórios
- Histórico de vendas
- Relatórios por período
- Controle de estoque
- Relatório de clientes
- Exportação para Excel (CSV)

### 💾 Backup e Exportação
- Backup completo do sistema
- Exportação em JSON
- Relatórios em CSV
- Recuperação de dados

### 📱 Integração com WhatsApp
- Envio de cobrança pelo WhatsApp
- Aviso de mensalidade vencida
- Aviso de vencimento próximo
- Contato direto com alunos

# ⚙️ Instalação

Clone o repositório:

```bash
git clone https://github.com/miguel14535/Arenajiu.git
```

Entre na pasta:

```bash
cd Arenajiu
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

Abra no navegador:

```text
http://localhost:5173
```

---

# 💾 Como Fazer Backup

1. Acesse a aba:

```text
Backup / Exportar
```

2. Clique em:

```text
Baixar Backup de Segurança
```

3. Salve o arquivo em:
- Google Drive
- OneDrive
- Pendrive
- HD Externo

# 🔐 Segurança

O sistema foi desenvolvido para uso local da academia.

Os dados permanecem armazenados no computador utilizado, permitindo:

- Maior velocidade
- Funcionamento offline
- Controle total dos dados

⚠️ Recomenda-se realizar backups periódicos.
